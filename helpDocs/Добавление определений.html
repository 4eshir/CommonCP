Для того чтобы добавить определение для нового термина необходимо:
1. Открыть файл с теоретическим материалом
  1.1 Перед определяемым термином добавить строку <a href = "#NAME"><a id = "NAME">, где NAME - это определяемый термин
  1.2 После определяемого термина добавить строку </a></a>
  1.3 Сохранить и закрыть файл
2. Открыть файл Словарь.html
  2.1 В конец файла добавить <a id = "NAME">
  2.2 На новую строку добавить определяемый термин
  2.3 На новой строке написать </a><br>
  2.4 С новой строки ввести определение термина
  2.5 На новую строку добавить сепаратор <br>----------<br>
  2.6 Сохранить и закрыть файл
